# Schema, vecka 5
###### JavaScript, vecka 4 av 8

## Introduktion

I denna modul kommer vi doppa fötterna framförallt i formulärvalidering, och då passar vi även på att trycka in felhantering som är ett centralt koncept inom all programmering, och som passar utmärkt när vi skall hantera de fel som formulärvalideringen hittar. Vi kommer även att ge oss på localStorage, som är webbläsarens eget sätt att spara ner data. Detta kommer hjälpa oss att låtsar om att vi interagerar med en databas redan nu, vilket kommer bli superanvändbart. Vi kommer även att introducera branscher i Git, och lägga grunden för att på sikt får till ett säkrare arbetssätt.

## Mål för veckan:

1. Förstå hur man interagerar med formulär i kod
2. Förstå vikten av formulärvalidering
3. Förstå hur felhantering fungerar
4. Förstå hur vi kan använda localStorage för att lagra data
5. Förstå hur vi samarbetar kring kod genom att använda oss av branscher i Git
